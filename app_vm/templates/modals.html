Information about the Application

This application has three parts.
  1. Database Tier
    A MongoDB database is serving as a backend database. Database server is accessible through REST API only. Details of the operations supported are accessible through "http://DB_VM_IP/docs" or "http://DB_VM_IP/redoc"
  2. Application Tier
    App server is running a Python Flask application and served by Gunicorn server. It also uses Datatables to showcase the table functionalities.
  3. Web Tier
    Web tier has Nginx as webserver. It is configured as reverse proxy and load balancer pointing to app server.
